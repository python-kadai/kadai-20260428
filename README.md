## 課題1：基本操作を使ってみよう

| No | 問題 | ヒント |
|----|------|--------|
| (1) | DataFrameの行数・列数を表示せよ | `df.shape` |
| (2) | 「好きな季節」列の中身を表示せよ | `df['列名']` |
| (3) | 睡眠時間が8時間の人だけ抽出せよ | `df[df['列名'] == 値]` |
| (4) | スマホ使用時間が5時間以下の人は何人いる？ | `len()` と組み合わせる |
| (5) | 朝食頻度ごとのストレスレベルの平均を求めよ | `groupby()` |

## 課題2：自由分析にチャレンジしよう

アンケートデータから **自分が面白いと思う問い** を作り、答えを出してみましょう。

**例：**
- 「好きな季節ごとにスマホ使用時間の平均は違う？」
- 「朝食を食べない人のストレスレベルは高い？」
- 「視力が悪い人はスマホをよく使っている？」

**ルール：**
- `groupby()` を必ず1回以上使うこと
- 結果を `print()` で表示すること
- 問いをコメントで書いてから答えを出すこと
